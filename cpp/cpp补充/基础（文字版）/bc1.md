二，常量与变量

1,常

（1）普通常量

数值，字符，字符串

默认：int，double**(floatl类型常量后加f，如1.23f)**

（2）转义字符

\\n换行

\\t水平制表符（按一下tab或四下空格）

**在程序中表示路径用\\\\表示\\**



2,变量

其值可以变化的量（数据） 先定义再使用

定义格式：

数据类型 变量名;（C语言中以分号作为结束）

如：

int age;

float height;

char sex;

（1）有符号变量

\- 有符号类型通过 signed 修饰，缺省 signed 默认是有符号



（2）无符号变量（非负数）

\- 只有整数类型能够声明为无符号的类型，无符号类型加上 unsigned 来修饰变量即可   比如队伍长度

\- 如:

&nbsp; - unsigned int age;

&nbsp; - unsigned char lv;
