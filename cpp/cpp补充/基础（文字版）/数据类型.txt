# C++里的"数据小盒子"（给小朋友的解释）

就像你有不同的玩具盒（装积木的、装汽车的、装娃娃的），C++里也有不同的"数据盒子"，用来装不同的东西～

## 1. 装数字的盒子
- **整数盒**（int）：只能装没有小数点的数，比如1、5、100（像你数的积木块数量）
  ```cpp
  int 苹果数量 = 3; // 装3个苹果
  ```

- **小数盒**（double）：可以装带小数点的数，比如3.14、1.5（像你喝的半杯果汁）
  ```cpp
  double 身高 = 1.2; // 装1.2米
  ```

- **超级大的数盒**（long long）：装特别大的数，比如1000000（像天上的星星数量）
  ```cpp
  long long 星星数量 = 1000000;
  ```


## 2. 装单个字的盒子（char）
像小印章一样，只能印一个字或符号，比如'a'、'3'、'#'（像你画画的单个小图案）
```cpp
char 第一个字母 = 'A'; // 装一个大写A
```


## 3. 装真假的盒子（bool）
只有两种可能：是（true）或不是（false）（像你点头说"要"或摇头说"不要"）
```cpp
bool 喜欢吃糖果 = true; // 真的喜欢
bool 天上有月亮 = false; // 假的，现在是白天
```


## 4. 装文字的盒子（string）
像小本子，可以写一整句话（像你画的连环画，一串图案连起来）
```cpp
string 名字 = "小明"; // 装一整个名字
string 一句话 = "我喜欢编程"; // 装一整句话
```


## 小总结
不同的盒子有不同的用处：
- 数个数用int，算小数用double
- 单个字母用char，一整句话用string
- 回答"是/不是"用bool

就像你不会把大象放进小盒子里，编程时也要选对盒子哦～
