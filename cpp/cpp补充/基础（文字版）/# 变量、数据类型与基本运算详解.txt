# 变量、数据类型与基本运算详解

## 一、变量：数据的"容器"

### 1. 什么是变量？
变量就像一个有名字的盒子，专门用来存放数据。你可以：
- 把数据放进盒子（赋值）
- 查看盒子里的内容（读取）
- 把盒子里的内容换成新的（修改）

### 2. 变量的基本规则
- 必须有一个唯一的名字（标识符）
- 名字只能包含字母、数字和下划线（_）
- 不能以数字开头
- 不能用编程语言的关键字（如C++中的int、if等）

### 3. 变量的使用步骤
```
1. 声明变量：告诉计算机"我需要一个叫XX的盒子"
2. 赋值：把数据放进盒子里
```

示例（C++风格）：
```cpp
int 年龄;      // 声明一个叫"年龄"的整数盒子
年龄 = 18;     // 把18放进这个盒子
int 身高 = 175; // 声明的同时赋值（更常用）
```


## 二、数据类型：给"盒子"分类

不同的数据需要不同类型的盒子来装，就像用不同的容器装水、装零食一样。

### 1. 数值类型
#### 整数类型
- **int**：最常用的整数盒子，能装-21亿到21亿之间的整数
  ```cpp
  int 苹果数量 = 5;
  int 温度 = -3;
  ```

- **long long**：装更大的整数（约-9e18到9e18）
  ```cpp
  long long 宇宙恒星数量 = 100000000000;
  ```

- **short**：小一点的整数盒子（节省空间）
  ```cpp
  short 班级人数 = 45;
  ```

#### 小数类型
- **double**：常用的小数盒子，精度较高
  ```cpp
  double 体重 = 62.5;   // 62.5公斤
  double 圆周率 = 3.1415926;
  ```

- **float**：精度较低的小数盒子（占用空间小）
  ```cpp
  float 商品价格 = 9.9;
  ```

### 2. 字符类型
- **char**：装单个字符的小盒子（字母、数字、符号）
  ```cpp
  char 性别 = '男';    // 注意用单引号
  char 等级 = 'A';
  ```

### 3. 布尔类型
- **bool**：只能装"是"或"否"的盒子
  ```cpp
  bool 及格 = true;    // true表示"是"
  bool 下雨 = false;   // false表示"否"
  ```

### 4. 字符串类型
- **string**：装一串字符（文字、句子）的盒子
  ```cpp
  string 名字 = "小明";  // 注意用双引号
  string 地址 = "北京市";
  ```


## 三、基本运算：操作数据的方法

### 1. 算术运算：和数学中的计算一样
| 运算符 | 作用 | 例子 | 结果 |
|--------|------|------|------|
| +      | 加   | 5 + 3 | 8 |
| -      | 减   | 5 - 3 | 2 |
| *      | 乘   | 5 * 3 | 15 |
| /      | 除   | 5 / 3 | 1（整数除法取整数部分） |
| %      | 取余 | 5 % 3 | 2（5除以3的余数） |

示例：
```cpp
int a = 10;
int b = 3;
int 和 = a + b;   // 13
int 商 = a / b;   // 3
int 余数 = a % b; // 1
```

### 2. 赋值运算：给变量装数据
| 运算符 | 作用 | 例子 | 等价于 |
|--------|------|------|--------|
| =      | 直接赋值 | a = 5 | - |
| +=     | 加后赋值 | a += 3 | a = a + 3 |
| -=     | 减后赋值 | a -= 3 | a = a - 3 |
| *=     | 乘后赋值 | a *= 3 | a = a * 3 |
| /=     | 除后赋值 | a /= 3 | a = a / 3 |

示例：
```cpp
int 分数 = 80;
分数 += 5;  // 分数变成85（相当于80 + 5）
分数 *= 2;  // 分数变成170（相当于85 × 2）
```

### 3. 自增自减运算：快速加减1
| 运算符 | 作用 | 例子 | 结果 |
|--------|------|------|------|
| ++     | 加1 | a++ | a的值增加1 |
| --     | 减1 | a-- | a的值减少1 |

示例：
```cpp
int 计数器 = 0;
计数器++;  // 计数器变成1
计数器++;  // 计数器变成2
计数器--;  // 计数器变成1
```


## 总结
- 变量是存放数据的"盒子"，有自己的名字
- 数据类型决定了"盒子"能装什么类型的数据
- 基本运算可以对数据进行计算和处理

这些是编程的基础中的基础，就像数学中的数字和加减乘除一样重要，掌握后才能进行更复杂的编程操作。
